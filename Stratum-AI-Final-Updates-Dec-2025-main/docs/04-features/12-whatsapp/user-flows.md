## Flow 1: Connect WhatsApp Business Account

**Actor**: Tenant Admin
**Goal**: Connect WhatsApp Business Cloud API

### Steps

1. User navigates to Settings â†’ WhatsApp
2. User clicks "Connect WhatsApp Business"
3. User enters Phone Number ID from Meta
4. User enters Business Account ID
5. User enters Access Token
6. User enters Webhook Verify Token
7. User clicks "Test Connection"
8. System validates credentials with Meta API
9. Connection saved, webhook URL displayed
