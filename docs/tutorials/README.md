## Available Tutorials

### [Starter Plan Tutorial](./starter-plan-tutorial.md)
**Estimated Time:** 30 minutes

Learn to use all Starter Plan features:
- Connecting ad platforms (Meta, Google, TikTok, Snapchat)
- Understanding the unified dashboard
- Signal health monitoring
- RFM customer analysis
- Dashboard exports
- Slack notifications

**Best for:** New users getting started with Stratum AI

---

### [Professional Plan Tutorial](./professional-plan-tutorial.md)
**Estimated Time:** 45 minutes

Master Professional Plan features (includes all Starter features):
- Funnel Builder for conversion analysis
- Computed Traits for dynamic segmentation
- Trust Gate Audit Logs
- Action Dry-Run mode
- Pipedrive CRM integration
- Advanced CDP segmentation

**Best for:** Growing teams ready to leverage automation and CRM integration

---

### [Enterprise Plan Tutorial](./enterprise-plan-tutorial.md)
**Estimated Time:** 60 minutes

Unlock the full power of Enterprise features (includes all Professional features):
- Consent Management (GDPR/CCPA)
- Real-Time Streaming events
- Predictive Churn modeling
- Custom Autopilot Rules
- Salesforce CRM integration
- LinkedIn Lead Gen integration
- Custom Dashboard Builder
- Custom Report Builder

**Best for:** Large organizations requiring advanced customization and compliance
