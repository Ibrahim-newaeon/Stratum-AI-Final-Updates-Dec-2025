## Flow 1: Connect Meta (Facebook) CAPI

**Actor**: Tenant Admin
**Goal**: Connect Meta Conversion API for server-side tracking

### Steps

1. User navigates to Settings â†’ Integrations
2. User clicks "Connect" on Meta card
3. Modal opens with credential inputs
4. User enters Pixel ID
5. User enters System User Access Token
6. User clicks "Test Connection"
7. System validates credentials with Meta API
8. Success: connection status updates to "Connected"
9. User can now send conversion events to Meta
