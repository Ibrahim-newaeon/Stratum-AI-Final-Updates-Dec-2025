6. Platform Connection Guide

Complete

What Users Need to Connect Each Platform

Each platform requires specific credentials and API access. Below are the complete requirements for connecting your ad accounts to Stratum AI.

Meta (Facebook/Instagram)

Required Credentials:
META_APP_ID=your-app-id META_APP_SECRET=your-app-secret META_ACCESS_TOKEN=system-user-token
OAuth Scopes:
  • ads_management - Create and manage ads
  • ads_read - Read ad data
  • business_management - Manage Business Manager
Setup Checklist:
  • Create Meta App at developers.facebook.com
  • Get App ID and App Secret
  • Create Facebook Pixel (get Pixel ID)
  • Create System User with permissions
  • Generate long-lived access token
Data Synced:

Campaigns, Ad Sets, Ads, Metrics, CAPI Events (Purchase, Lead, AddToCart)

Status: Production Ready

Google Ads

Required Credentials:
GOOGLE_ADS_CLIENT_ID=your-client-id GOOGLE_ADS_CLIENT_SECRET=your-client-secret GOOGLE_ADS_DEVELOPER_TOKEN=your-dev-token GOOGLE_ADS_REFRESH_TOKEN=from-oauth
OAuth Scopes:
  • adwords - Full Google Ads API access
Setup Checklist:
  • Create OAuth app in Google Cloud Console
  • Apply for Developer Token (needs business verification)
  • Complete OAuth flow for refresh token
  • Get Customer ID (without dashes)
  • Create Conversion Action in Google Ads
Data Synced:

Campaigns, Ad Groups, Keywords, Metrics, Enhanced Conversions, GA4 Integration

Status: Production+ (Advanced)

TikTok Ads

Required Credentials:
TIKTOK_APP_ID=your-app-id TIKTOK_APP_SECRET=your-app-secret
OAuth Scopes:
  • advertiser.read/write - Advertiser access
  • campaign.read/write - Campaign management
  • report.read - Access to reports
Setup Checklist:
  • Create TikTok Business Account
  • Apply for Marketing API access
  • Get App ID and App Secret
  • Create TikTok Pixel (get Pixel Code)
  • Generate long-lived access token
Data Synced:

Campaigns, Ad Groups, Ads, Events API (CompletePayment, AddToCart, ViewContent)

Status: Production Ready

Snapchat Ads

Required Credentials:

OAuth Scopes:
  • snapchat-marketing-api - Full Marketing API
Setup Checklist:

  • Register OAuth app with Snapchat

7. Trust Engine & Autopilot

Critical System

Signal Health Thresholds

Status Score Range Color Autopilot Mode
HEALTHY 70-100 Green Normal - Full autopilot enabled
RISK 60-69 Yellow Limited - Conservative actions only
DEGRADED 40-59 Orange Cuts Only - Pause/reduce only
CRITICAL 0-39 Red Frozen - No automation

Signal Health Components

EMQ Score (40% weight)

Event Match Quality from platform data. Measures conversion event reliability.

Freshness Score (25% weight)

Data recency. <24h = 100%, 24-48h = decay, >48h = 0%

Variance Score (20% weight)

Platform vs GA4 variance. Warning at 10%, critical at 15%.

Anomaly Score (15% weight)

Z-score analysis on spend, conversions, CPA, ROAS.

Security Concerns

Severity Issue Recommendation
Critical Hardcoded database credentials in config.py defaults Remove all default passwords; use env vars only
Critical Weak default JWT/PII encryption keys Enforce minimum 32-byte keys in production
High Fixed PII encryption salt Implement per-tenant salt strategy
High CORS allows all methods/headers Restrict to specific origins in production
Medium No HTTPS/HSTS enforcement Add Strict-Transport-Security header
Medium WebSocket auth token in query params Move to Authorization header
